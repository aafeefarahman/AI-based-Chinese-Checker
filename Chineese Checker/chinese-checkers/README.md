# Chinese Checkers - Full Stack Game

## Quick Start

### 1. Start Backend (Terminal 1)
```
cd backend
python app.py
```
Runs on http://localhost:5000

### 2. Start Frontend (Terminal 2)
```
cd frontend
npm run dev
```
Runs on http://localhost:3000

## Tech Stack
- Frontend: React + Vite + CSS
- Backend: Flask + SQLite
- AI: Minimax with Alpha-Beta Pruning

## Game Modes
- Human vs AI: You play as Purple (top), AI plays as Green (bottom)
- AI vs AI: Watch two AIs battle it out automatically
